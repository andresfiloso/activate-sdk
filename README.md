# activate-sdk

## Install

```bash
npm install activate-sdk
```

## Usage

#### Typescript
```ts
import ActivateSDK from 'activate-sdk';

ActivateSDK.helpers.calculateUnitCost(50, [])

```

#### Javascript
```js
const ActivateSDK = require('activate-sdk');

ActivateSDK.helpers.calculateUnitCost(50, [])
```
